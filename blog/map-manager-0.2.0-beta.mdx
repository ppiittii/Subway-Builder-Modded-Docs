You can download the latest release of the Subway Builder Map Manager at: https://github.com/Kronifer/subwaybuilder-patcher/releases/tag/v0.2.0-beta. Update your Map Manager before proceeding with any guide.

### Changelog:

#### Features:
 - More than one map can be selected for install at a time
 - If a map doesn't include a thumbnail but includes thumbnailBbox in its config, the manager will autogenerate a thumbnail

#### Fixes:
 - Fixed an issue where vanilla maps could be overwritten by the map manager
 - Fixed an issue where the map manager attempted to download the wrong version of pmtiles for Linux

If anyone finds any bugs, please post them in the [issues tab](https://github.com/Kronifer/subwaybuilder-patcher/issues) with the prefix mod_beta or send them to me in the thread on [Discord](https://discord.gg/jrNQpbytUQ).
