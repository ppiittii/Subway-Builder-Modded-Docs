title: Anleitung zur Karteninstallation
---

[Kronifer's Map Manager](https://github.com/Kronifer/subwaybuilder-patcher/releases) ist die einfachste Möglichkeit, benutzerdefinierte Städte in Subway Builder zu installieren.

### Schritt 1 - Map Manager installieren

Lade die neueste Version herunter.

:::note
Windows: `.exe`  
Linux: `.zip`  
MacOS: `arm64.dmg` (Apple Silicon) oder `x64.dmg` (Intel)
:::

:::note
Unter MacOS kann die .dmg-Datei in Quarantäne sein. Führe aus:
`xattr -d com.apple.quarantine {INSTALLER_FILENAME}`
:::

Installiere die App normal und wähle die Spiel-Exe sowie den `metro-maker4`-Ordner.

### Schritt 2 - Karte hinzufügen

Klicke auf `ADD A MAP` und importiere die ZIP-Datei.

### Schritt 3 - Subway Builder starten

Starte das Spiel mit `START GAME`. Karten erscheinen im Tab `CUSTOM`.

:::important
Das Spiel muss jedes Mal über den Map Manager gestartet werden, damit Gebäude und Wasser korrekt angezeigt werden.
:::
