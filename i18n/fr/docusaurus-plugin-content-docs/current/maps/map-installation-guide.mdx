title: Guide d'installation des cartes
---

[Kronifer's Map Manager](https://github.com/Kronifer/subwaybuilder-patcher/releases) est le moyen le plus simple et le plus efficace de télécharger et d’installer des villes personnalisées dans Subway Builder. Tout ce dont vous avez besoin est un fichier ZIP fourni ainsi que le Map Manager lui-même.

### Étape 1 - Installer le Map Manager

Téléchargez la dernière version du Map Manager sur https://github.com/Kronifer/subwaybuilder-patcher/releases. Cliquez sur la version correspondant à votre système d’exploitation.

:::note
Sous Windows, utilisez le fichier `.exe`. Sous Linux, utilisez le fichier `.zip`. Sous MacOS, utilisez le fichier `arm64.dmg` si votre Mac possède une puce Apple Silicon (M1, M2, M3, etc.) et le fichier `x64.dmg` s’il possède une puce Intel. En règle générale, les Mac après 2021 utilisent Apple Silicon, mais vérifiez car il existe des exceptions.
:::

:::note
Sous MacOS, le fichier .dmg peut être mis en quarantaine. Pour l’exécuter, ouvrez un terminal dans le dossier de téléchargement et exécutez `xattr -d com.apple.quarantine {INSTALLER_FILENAME}`. Par exemple : `xattr -d com.apple.quarantine Subway.Builder.Map.Manager-0.2.7-arm64.dmg`.

Le dossier Library contenant le dossier metro-maker4 peut être masqué par défaut. Ouvrez Finder → dossier Home → View → Show View Options → Show Library Folder.
:::

Installez ensuite l’application normalement. Après l’installation, lancez l’application et configurez-la en sélectionnant l’exécutable du jeu et le dossier `metro-maker4`.

### Étape 2 - Ajouter la carte

Cliquez sur `ADD A MAP` et importez le fichier ZIP téléchargé. Le nom de la carte, le créateur et la version apparaîtront à l’écran.

### Étape 3 - Lancer Subway Builder

Après avoir ajouté toutes vos cartes, lancez le jeu avec `START GAME`. Les cartes apparaîtront dans l’onglet `CUSTOM`.

:::important
Vous devez lancer le jeu via le Map Manager à chaque fois pour que les bâtiments et l’eau apparaissent correctement.
:::
