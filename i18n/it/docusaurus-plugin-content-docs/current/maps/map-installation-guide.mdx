title: Guida all'installazione delle mappe
---

[Kronifer's Map Manager](https://github.com/Kronifer/subwaybuilder-patcher/releases) è il modo più semplice e veloce per scaricare e installare città personalizzate in Subway Builder.

### Passaggio 1 - Installare il Map Manager

Scarica l’ultima versione da https://github.com/Kronifer/subwaybuilder-patcher/releases.

:::note
Su Windows usa `.exe`, su Linux `.zip`, su MacOS `arm64.dmg` (Apple Silicon) o `x64.dmg` (Intel).
:::

:::note
Su MacOS il file .dmg potrebbe essere in quarantena. Esegui:
`xattr -d com.apple.quarantine {INSTALLER_FILENAME}`
:::

Installa normalmente e configura selezionando l’eseguibile del gioco e la cartella `metro-maker4`.

### Passaggio 2 - Aggiungere la mappa

Clicca su `ADD A MAP` e importa il file ZIP.

### Passaggio 3 - Avviare Subway Builder

Usa `START GAME` per avviare il gioco. Le mappe appariranno nella scheda `CUSTOM`.

:::important
Devi sempre avviare il gioco tramite il Map Manager per far apparire correttamente edifici e acqua.
:::
